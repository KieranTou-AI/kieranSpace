## 为什么搭建这个博客

这是我的个人技术博客，用来记录学习和实践过程中的知识点。

![kieranSpace](/images/placeholder.png)

## 技术栈

这个博客采用了以下技术：

- **Next.js** — 静态站点生成，零服务器
- **Tailwind CSS + shadcn/ui** — 样式和组件
- **MDX** — 文章格式，支持在 Markdown 中嵌入 React 组件
- **Vercel** — 部署平台

## 代码示例

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("kieranSpace"));
```

## 工作流程

1. 在 `content/` 目录编写 `.mdx` 文件
2. `git push` 推送到 GitHub
3. Vercel 自动构建部署
