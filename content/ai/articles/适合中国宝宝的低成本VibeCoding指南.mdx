## 基础准备

### 软件

**代码编辑器VSCode**

下载相应VSCode的Extensions（拓展）：
- MDX Preview（在VSCode中预览mdx文件）
- Chinese (Simplified) (简体中文)
- Claude Code for VS Code

安装完成后，重启VSCode。右上角点击太阳花图标

![显示失败](/images/vibecoding2.png)

即可显示：

![显示失败](/images/vibecoding1.png)

**Git**

使用Github 用于后续下载cc-switch

![显示失败](/images/vibecoding3.png)

> 作者Github地址：https://github.com/KieranTou-AI

**node/npm**

点击下载对应版本的[node.js和npm](https://nodejs.org/zh-cn/download)

安装完成后在终端中输入以下命令检查是否安装成功：

```bash
node --version
npm --version
```

**Claude**

通过npm安装Claude

```bash
npm install -g @anthropic-ai/claude-code
claude -v
```
> 注意：如果在安装过程中遇到权限问题，请尝试使用`sudo`命令。

### 大模型

**DeepSeek API Key**

1. 进入[deepseek开放平台](https://platform.deepseek.com/usage)
2. 注册&充值
3. 获取API Key

## 实现关联

### Github下载对应版本的cc-switch

[点击下载cc-switch](https://github.com/farion1231/cc-switch/releases)

### 配置cc-switch

1. 在命令行Claude界面点击右上角的添加，选择DeepSeek。
2. 输入DeepSeek API Key。
![显示失败](/images/vibecoding4.png)
3. 配置模型映射，[参考文档](https://api-docs.deepseek.com/zh-cn/quick_start/agent_integrations/claude_code)。
我是这么配的：![显示失败](/images/vibecoding5.png)

4.配置JSON处选择最大强度思考。
